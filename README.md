## 🚀 Getting Started

### Installation 👨🏻‍💻

1. Install all packages

```
npm i
```

2. Run Development Server

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. Build your project

```
npm run build
```
